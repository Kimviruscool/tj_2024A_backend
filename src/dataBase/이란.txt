[ 컴퓨터 ]
    [메모리 종류]
    - 주기억장치         : 메모리(RAM)
        - 실행중인 메모리 관리 (휘발성) , 컴퓨터가 꺼지면 초기화/사라짐.
        - JVM
        - 실행중인 프로그램내 메모리만 저장하기때문에 영구저장이 불가능하다.

    - 보조기억장치       : HDD , SSD (ROM)
        - 비실행중인 메모리 관리 (비휘발성) , 컴퓨터가 꺼져도 저장이가능.
        - 파일(메모장) 처리 , 데이터베이스(DB)
        - C드라이브 , usb , cd 등등 : 내가 직접 삭제 하기 전 까지 영구 저장


자바는 JVM (메모리 관리) 이라는 구조로 메모리 관리 (휘발성) , 연구저장 불가능

데이터베이스 : 엑셀 , 엑세스 , 한글 등등 데이터베이스 기반 프로그램
데이터베이스란?
    1. 데이터/자료 란? : 순수한 자료
    2. 베이스 : 모임/집합
    --------------> 자료들의 집합/모음

데이터베이스 종류
1. 트리형
2. 네트워크형
3. 관계형
4. NoSQL
학습 & 실무 : 관계형 데이터베이스
1. 관계형데이터베이스 : 행과열로 이루어진 데이터의 집합 ==> 표/테이블 (엑셀)
2. 표/테이블 고유값/식별 값을 참조하여 서로 다른 테이블 끼리 종속되는 관계를 표현 하는 구조


관계형 데이터베이스 사용하는 프로그램
    1. Oracle 2.Mysql 3.MariaDB 등등

알아야 하는 용어
DB : 데이터베이스(데이터의 집합 ==> 표/테이블)
DBA : DBA 데이터베이스 관리자 (DATABASE ADMIN)
DBMA : 데이터베이스 관리시스템 (1. Oracle 2.Mysql 3.MariaDB 등등)
DB SERVER : 데이터베이스가 설치된 PC
    - SERVER : 관리자/본인 자원을 제공 하는
    - CLIENT : 사용자/고객 자원을 요청 하는

SQL : 데이터베이스 시스템 프로그램이 이해할수 있는 언어 --> 문법

ex) 가계부 표 만들어줘 ===========> create table 가계부{}
